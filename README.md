> Fully autonomous endpoint security agent simulated in C to detect unauthorized system behaviors, process anomalies, and signature-based threats.

| Attribute | Detail |
|---|---|
| **Core Stack** | C · POSIX APIs |
| **Mechanisms** | Process Monitoring · Static Signature Matching · Real-time Log Generation |
| **Architecture** | Lightweight standalone single-executable |
| **Repository** | *Simulated Core Architecture* |

**What it does:** Runs as a continuous background utility to track simulated process behaviors. Employs lightweight pattern-matching and systems calls to catch baseline anomalies with minimal resource overhead.

<details>
<summary><b>📊 Security Log Analyzer — Automated Parser</b></summary>

<br/>

> High-performance, C-based parsing engine built to ingest raw system logs, detect malicious patterns, and structure security analytics.

| Attribute | Detail |
|---|---|
| **Core Stack** | C · File I/O · String Manipulation · Git |
| **Mechanisms** | Log Filtering · Severity Matrix · Indicator of Compromise (IoC) Alerts |
| **Output** | Structured Security Event Generation (TXT/CSV Format) |

**What it does:** Ingests raw textual log files (Syslog/Auth logs), breaks them down using pointer-based tokenization, and automatically isolates high-risk entries to reduce manual threat investigation time.

</details>
